About Me

Saya Muhammad Hana Haikal, lulusan D3 Teknik Informatika dari Politeknik Purbaya. Saya memiliki antusiasme yang tinggi dalam dunia pemrograman dan teknologi, serta berkomitmen untuk terus mengembangkan diri di bidang ini.

Selama 3 tahun studi, saya mempelajari dasar-dasar pemrograman dengan PHP, JavaScript, dan HTML, serta memahami penggunaan tools version control system seperti Git dan GitHub. Selain itu, saya juga memiliki pemahaman tentang pengelolaan database menggunakan MySQL yang mendukung kemampuan teknis saya.

Meskipun pengalaman saya di dunia kerja belum banyak, saya yakin dengan karakter yang mudah beradaptasi, cepat tanggap terhadap hal baru, serta dukungan dari komunitas dan rekan-rekan, saya dapat terus berkembang. Saya percaya bahwa ilmu bisa datang dari mana saja, asalkan ada niat dan minat yang kuat untuk menjalaninya.
